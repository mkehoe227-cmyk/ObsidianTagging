# Atomic Notes and Evergreen Writing

An atomic note contains exactly one idea. This constraint forces clarity and enables flexible linking across different contexts. Andy Matuschak's evergreen notes philosophy emphasizes that notes should be written for future use, not just for reference. Each note should be a self-contained unit that stands on its own while connecting to the broader knowledge network through links and tags.
