# The Zettelkasten Method

The Zettelkasten method is a [[Personal Knowledge Management Systems|personal knowledge management system]] developed by sociologist Niklas Luhmann. It emphasizes creating atomic, interlinked notes rather than hierarchical folders. Each note contains a single idea and links to related concepts, forming an emergent network of knowledge.
